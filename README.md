# SharagaProject — Магазин электроники (Android App)

Учебное мобильное приложение для заказа товаров, разработанное на языке **Kotlin** в рамках обучения Android-разработке.

## 📱 Основной функционал

*   **Авторизация и регистрация** (Система входа пользователей. Данные сохраняются локально.)
*   **Каталог товаров**
*   **Корзина** (Возможность добавлять товары в корзину для последующего оформления заказа.)
*   **История заказов.** (Activity: `HistoryActivity`) Просмотр списка всех совершенных покупок конкретным пользователем.
*   **Локальная база данных.** Использование SQLite (через `DBHelper`) для хранения данных о пользователях, товарах и заказах.
*   **Сохранение сессии.** Использование `SharedPreferences` для того, чтобы пользователь оставался залогиненным при перезапуске приложения.

## 🛠 Технологический стек

*   **Язык программирования:** [Kotlin](https://kotlinlang.org/)
*   **UI Framework:** XML (Material Design)
*   **Компоненты Android Jetpack:**
    *   `RecyclerView` — для отображения списков заказов и товаров.
    *   `AppCompatActivity` — базовая логика экранов.
    *   `ConstraintLayout` — современная адаптивная верстка.
*   **Хранение данных:**
    *   `SQLite` — полноценная реляционная БД для истории и пользователей.
    *   `SharedPreferences` — хранение ID текущего пользователя.

## 📂 Структура проекта (основные классы)

*   `MainActivity.kt` — Главный экран приложения.
*   `HistoryActivity.kt` — Экран истории покупок (связан с БД).
*   `HistoryAdapter.kt` — Адаптер для отображения элементов списка в RecyclerView.
*   `DBHelper.kt` — Класс для работы с базой данных (создание таблиц, CRUD операции).
